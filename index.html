HPC Account Provisioning Automation

View Repository

A thesis project that develops an HPC account provisioning automation system using Flask, Ansible, and Docker. Includes performance testing (execution time, TTFA, p95 latency) and resource monitoring using Prometheus–Grafana. The system emphasizes idempotency, scalability, and observability in HPC environments.
